## 完成的工作

1. **创建了 `tools` 目录**，用于存放脚本工具
2. **实现了 `va_index_calculator.sh` 脚本**，功能包括：
   - 输入 64 位虚拟地址（十六进制格式）
   - 输出 L0、L1、L2、L3 各级索引值
   - 使用纯 bash 实现，不依赖外部命令
   - 自动处理地址格式和位计算

## 测试结果

### 测试 1：`0xFFFF800040000000`
```
L0 索引 (bit 47-39): 256
L1 索引 (bit 38-30): 1
L2 索引 (bit 29-21): 0
L3 索引 (bit 20-12): 0
```

### 测试 2：`0xFFFF800040100000`
```
L0 索引 (bit 47-39): 256
L1 索引 (bit 38-30): 1
L2 索引 (bit 29-21): 0
L3 索引 (bit 20-12): 256
```

## 使用方法

```bash
./tools/va_index_calculator.sh <虚拟地址>
```

例如：
```bash
./tools/va_index_calculator.sh 0xFFFF800040000000
```

这个工具可以帮助快速计算 ARM64 虚拟地址的各级索引值，对于调试和理解页表结构非常有用！
