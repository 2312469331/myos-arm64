

## 一、printk 16进制格式符（ARM64 内核必用）
| 格式符 | 作用 | 输出示例 | 带0x？ |
|--------|------|----------|--------|
| `%x`   | 32位16进制（小写） | `123abc` | ❌ 否 |
| `%X`   | 32位16进制（大写） | `123ABC` | ❌ 否 |
| `%lx`  | **64位16进制（小写）** | `ffff000012345678` | ❌ 否 |
| `%lX`  | 64位16进制（大写） | `FFFF000012345678` | ❌ 否 |
| `%#x`  | 32位 + 自动加 `0x` | `0x123abc` | ✅ 是 |
| `%#lx` | **64位 + 自动加 `0x`**（你最常用） | `0xffff000012345678` | ✅ 是 |
